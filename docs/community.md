# 社区

欢迎来到 Abaqus 中文文档社区！这里，您可以找到与文档相关的讨论、贡献指南和资源链接。

## 参与贡献

我们欢迎任何形式的贡献，包括但不限于：

- :material-translate: **翻译改进** - 优化中文表达或补充遗漏内容
- :material-bug: **错误报告** - 发现文档中的错误请提交 Issue
- :material-file-document-edit: **内容完善** - 添加示例或澄清模糊说明
- :material-translate-variant: **新语言** - 帮助创建其他语言版本

## 如何贡献

### 步骤 1：Fork 仓库

访问我们的 [GitHub 仓库](https://github.com/example/guide-of-abaqus)，点击 Fork 按钮创建您自己的副本。

### 步骤 2：克隆本地

```bash
git clone https://github.com/YOUR_USERNAME/guide-of-abaqus.git
cd guide-of-abaqus
```

### 步骤 3：创建分支

```bash
git checkout -b feature/improve-docs
```

### 步骤 4：提交更改

```bash
git commit -m "描述您的更改"
git push origin feature/improve-docs
```

### 步骤 5：创建 Pull Request

在 GitHub 上创建 Pull Request，我们会尽快审核。

## 讨论渠道

| 渠道 | 用途 |
|------|------|
| GitHub Issues | Bug 报告、功能请求 |
| GitHub Discussions | 常规讨论、问答 |
| 邮件列表 | 重要公告更新 |

## 行为准则

请遵守以下准则，保持社区友好和专业：

- 尊重所有参与者
- 使用包容性语言
- 专注于建设性反馈
- 对事不对人
