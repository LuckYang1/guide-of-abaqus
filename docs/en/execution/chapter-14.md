# File Extensions

## File Extension Definitions

**Products**: Abaqus/Standard Abaqus/Explicit

### References

- About Execution Procedures

### Overview

The Abaqus program generates multiple files. Some of these files contain analysis, postprocessing, and translation results and are kept for use by other analysis options, restarts, or postprocessing. This section describes files that Abaqus creates and keeps.

Other files exist only during Abaqus execution and are deleted when the run completes. These temporary files are generated in the scratch directory.

### File Extensions

#### .abq

State file, used only by Abaqus/Explicit.

#### .axi

Symmetric model data file, used only by Abaqus/Standard.

#### .bsp

Text file containing beam section properties.

#### .c

User subroutine or other special-purpose C file.

#### .cax

File containing information required for Abaqus/Standard to continue data check runs.

#### .cid

Auto-release file, containing information needed for license recovery and suspension.

#### .com

Command file, created by the Abaqus execution procedure.

#### .dat

Print output file.

#### .f

User subroutine or other special-purpose Fortran file.

#### .fil

Results file.

#### .fin

Results file created when changing .fil file format using the abaqus ascfil command.

#### .inp

Analysis input file.

#### .ipm

Inter-process message file.

#### .lck

Lock file for output database.

#### .log

Log file.

#### .mdl

Model file.

#### .msg

Message file.

#### .nck

Nickname file.

#### .odb

Output database.

#### .pac

Package file, used only by Abaqus/Explicit.

#### .par

Modified version of a parametric input file.

#### .pes

Modified version of a parametric input file.

#### .pmg

Parameter evaluation and substitution message file.

#### .prt

Part file.

#### .psf

Python script file.

#### .res

Restart file.

#### .sel

Selected results file, used only by Abaqus/Explicit.

#### .sim

Model and results file.

#### .sta

Status file.

#### .stt

Status file.

#### .sup

Substructure file.

#### .var

File containing input file variant information generated by parametric studies.

#### .023

Communication file.
