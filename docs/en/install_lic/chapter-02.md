# Chapter 2: What's New

This page describes the latest changes in Abaqus Configuration.

## 2025 FD01

### PDF Guides Available

You can download PDF versions of the Abaqus guides from each guide overview page.

**Benefits**: You can easily access the PDF versions of the guides.

In earlier releases, PDF versions of the guides were updated only for the GA (General Availability) release and were available for download from the Dassault Systèmes Knowledge Base. Starting with 2025 FD01, PDF guides are updated alongside the HTML version guides.

## 2024 FD01

### threads_per_mpi_process Parallelization Parameter

The `threads_per_mpi_process` parameter can be used to select among three different parallel execution modes: pure MPI (DMP), pure thread (SMP), and a hybrid mode combining MPI and threads (HMP).

**Benefits**: The new parameter helps users customize parallel execution in Abaqus.
