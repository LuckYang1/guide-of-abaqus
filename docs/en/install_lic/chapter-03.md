# Chapter 3: Abaqus Configuration Guide

The Abaqus Configuration Guide contains information for customizing Abaqus 2025.

Additionally, the guides in the Abaqus&reg; documentation set describe the capabilities of the Abaqus finite element analysis technology used in SIMULIA applications.

For instructions on the installation procedure, refer to the SIMULIA Installation Guide.

Environment files provide options for customizing an Abaqus installation. Specifying the file system to be used for temporary files is particularly important. For detailed information on how to set local defaults and tune your system configuration for running Abaqus jobs efficiently, refer to Customizing the Abaqus Environment.

For more information on customizing the Abaqus user interface, printer configuration, and graphics card tuning, refer to Customizing Abaqus/CAE and Abaqus/Viewer.
