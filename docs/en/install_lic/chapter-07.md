# Chapter 7: Installation Subdirectories

SIMULIA products are installed into the same directory.

The following subdirectories are created:

| Subdirectory | Contents Description |
|--------------|---------------------|
| `install_dir/os/CAEresources/` | Configuration files |
| `install_dir/os/code/bin/` | SIMULIA product executables, third-party executables, and libraries |
| `install_dir/os/code/command/` | SIMULIA product command procedures |
| `install_dir/os/code/include/` | Header file links for building postprocessors using the abaqus make utility |
| `install_dir/os/SMA/samples/` | Input files and directories associated with installed products, including installation verification problems, timing test problems, Introduction to Abaqus workshop files, Getting Started tutorial guide files, application brief description files, files for use with the abaqus findkeyword utility, and files from the Example Problems, Benchmarks, and Verification Guides |
| `install_dir/os/SMA/site/` | Site-specific files: Abaqus environment files (`abaqus_v6.env` and `custom_v6.env`), example environment file (`abaqusinc.env`), include files (`file_name.inc`), data files containing information used by verification procedures (`chksum.dat`), and platform computer tables |
| `install_dir/os/tools/SMApy` | Python interpreter |
| `install_dir/os/code/python2.7/lib/` | Abaqus Python modules |
