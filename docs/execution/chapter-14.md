# Fortran 单元号

## Fortran 单元号

**产品**：Abaqus/Standard、Abaqus/Explicit

### 参考文献

- 关于执行程序

### 概述

Abaqus 使用下表中概述的 Fortran 单元号。除非另有说明，否则不应尝试从用户子程序写入这些 Fortran 单元。

对于 Abaqus/Standard，您应该指定单元号 15-18 或大于 100 的单元号。对于 Abaqus/Explicit，请指定单元 16-18 或大于 100 且以 5 到 9 结尾的单元号，例如 105、268 等。您不能写入 .sta 文件。

### Fortran 单元号

#### Abaqus/Standard 单元号

| 单元号 | 描述 |
|-------|------|
| 1 | 内部数据库 |
| 2 | 求解器文件 |
| 6 | 打印输出 (.dat) 文件（您可以写入此文件） |
| 7 | 消息 (.msg) 文件（您可以写入此文件） |
| 8 | 结果 (.fil) 文件 |
| 10 | 内部数据库 |
| 12 | 重启 (.res) 文件 |
| 19-30 | 内部数据库（临时文件）。单元号 21 和 22 始终写入磁盘 |
| 73 | 包含网格梁截面属性 (.bsp) 的文本文件 |

#### Abaqus/Explicit 单元号

| 单元号 | 描述 |
|-------|------|
| 6 | 打印输出 (.log) 文件 |
| 12 | 重启 (.res) 文件 |
| 13 | 旧重启 (.res) 文件（如果适用） |
| 15 | 分析预处理器 (.dat 或 .pre) 文件 |
| 23 | 通信 (.023) 文件 |
| 60 | 全局包 (.pac) 文件 |
| 61 | 全局状态 (.abq) 文件 |
| 62 | 临时文件 |
| 63 | 全局选择结果 (.sel) 文件 |
| 64 | 消息 (.msg) 文件 |
| 65 | 输出数据库 (.odb) 文件 |
| 67 | 旧包 (.pac) 文件（如果从 Abaqus/Explicit 导入） |
| 68 | 旧状态 (.abq) 文件（如果从 Abaqus/Explicit 导入） |
| 69 | 内部数据库；临时文件 |

#### 域并行单元号

如果使用域并行：

| 单元号 | 描述 |
|-------|------|
| 70 | CPU #1 的本地包 (.pac.1) 文件 |
| 71 | CPU #1 的本地状态 (.abq.1) 文件 |
| 73 | CPU #1 的本地选择结果 (.sel.1) 文件 |
| 80 | CPU #2 的本地包 (.pac.2) 文件 |
| 81 | CPU #2 的本地状态 (.abq.2) 文件 |
| 83 | CPU #2 的本地选择结果 (.sel.2) 文件 |

每个额外的 CPU 添加三个文件，单元号递增 10。
