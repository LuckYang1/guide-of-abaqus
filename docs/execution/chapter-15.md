## 快速参考

### 常用文件扩展名

| 扩展名 | 描述 |
|-------|------|
| .inp | 分析输入文件 |
| .odb | 输出数据库 |
| .dat | 打印输出 |
| .log | 日志文件 |
| .res | 重启文件 |
| .fil | 结果文件 |
| .stt | 状态文件 |
| .mdl | 模型文件 |
| .msg | 消息文件 |
| .abq | 状态文件 (Abaqus/Explicit) |
| .pac | 包文件 (Abaqus/Explicit) |
| .sel | 选择结果文件 (Abaqus/Explicit) |

### 常用 Fortran 单元号

**Abaqus/Standard**：
- 单元 6：打印输出
- 单元 7：消息文件
- 单元 8：结果文件
- 单元 12：重启文件

**Abaqus/Explicit**：
- 单元 6：打印输出
- 单元 12：重启文件
- 单元 65：输出数据库
