# Abaqus GUI Toolkit 用户指南

## 文档概述

本指南描述了 Abaqus GUI Toolkit，它允许您通过创建自己的对话框、GUI 模块甚至完整的应用程序来定制 Abaqus/CAE 图形用户界面。本指南旨在引导您完成编写应用程序的过程，解释如何使用工具包的组件并提供示例代码片段。

本指南是 Abaqus 文档集合的一部分，描述了用于 SIMULIA 应用程序的 Abaqus 有限元分析技术的所有功能。

## 技术术语说明

本指南中保留了大量技术术语的英文原名，以确保与 Abaqus 官方文档的一致性和准确性。主要术语包括：

- **Kernel（内核）** - Abaqus/CAE 的核心进程，负责数据库访问和命令执行
- **GUI** - 图形用户界面（Graphical User Interface）
- **Widget（控件）** - GUI 中的用户界面元素，如按钮、文本框等
- **Mode（模式）** - 收集用户输入并向内核发送命令的机制
- **Module（模块）** - 功能分组，如 Part 模块
- **Toolset（工具集）** - 类似模块但更具体的功能分组
- **Dialog Box（对话框）** - 用于收集用户输入的窗口
- **Layout Manager（布局管理器）** - 排列对话框中控件的机制
