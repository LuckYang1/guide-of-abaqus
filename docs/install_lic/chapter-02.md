# 第二章：新增内容

本页面描述 Abaqus Configuration 的最新变更。

## 2025 FD01

### PDF 指南可用

您可以从每个指南概述页面下载 Abaqus 指南的 PDF 版本。

**优势**：您可以轻松访问指南的 PDF 版本。

在早期版本中，指南的 PDF 版本仅在 GA（通用版本）发布时更新，并可从 Dassault Systèmes 知识库下载。从 2025 FD01 开始，PDF 指南将与 HTML 版本指南一起更新。

## 2024 FD01

### threads_per_mpi_process 并行化参数

`threads_per_mpi_process` 参数可用于在三种不同的并行执行模式之间进行选择：纯 MPI (DMP)、纯线程 (SMP) 以及 MPI 和线程的混合模式 (HMP)。

**优势**：新参数帮助用户自定义 Abaqus 中的并行执行。
