# 第三章：Abaqus 配置指南

Abaqus Configuration Guide 包含自定义 Abaqus 2025 的信息。

此外，Abaqus&reg; 文档集合中的指南描述了 SIMULIA 应用程序中使用的 Abaqus 有限元分析技术的能力。

有关安装程序的说明，请参阅 SIMULIA Installation Guide。

环境文件提供了自定义 Abaqus 安装的选项。指定用于存储临时文件的文件系统尤为重要。有关如何设置本地默认值和调整系统配置以高效运行 Abaqus 作业的详细信息，请参阅自定义 Abaqus 环境。

有关自定义 Abaqus 用户界面、打印机配置和图形卡调优的更多信息，请参阅自定义 Abaqus/CAE 和 Abaqus/Viewer。
