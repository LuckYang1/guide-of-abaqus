## 第 4 章 - 查找更多信息

在仔细阅读本手册之后，您应该已经初步了解如何高效地使用 CST Studio Suite 进行粒子动力学仿真，以解决自己的问题。不过，在创建第一个自己的模型时，仍然会遇到许多问题。本章将简要介绍可用的文档资源。

## QuickStart Guide（快速入门指南）

QuickStart Guide 的主要作用，是提醒您完成成功执行仿真所需的所有必要步骤。尤其是对于新用户，或者很少使用该软件的用户，这种辅助会很有帮助。

如果勾选了 File: Options  Preferences  Open QuickStart Guide on project load 复选框，则每次项目启动时都会自动打开 QuickStart Guide。您也可以随时通过右上角 Help 按钮中的 QuickStart Guide 启动此助手。

启动 QuickStart Guide 后，会打开一个对话框，其中显示任务列表；每一项都代表模型定义和仿真过程中的一个步骤。通常，项目模板会预先设置问题类型，并初始化一些基本设置，例如单位和背景属性。否则，QuickStart Guide 会首先打开一个对话框，您可以在其中指定希望分析的计算类型，然后单击 Next 按钮继续：

![](../images/dd4550f1cfbb92a76b9b19f84bded9d465a890239c6d3875bd4b10d3d309c2ff.jpg)

一旦您成功完成某个步骤，相应条目就会被勾选，下一个必要步骤会被高亮显示。不过，在整个流程中，您仍然可以修改之前的任何设置。

若要访问有关 QuickStart Guide 本身的信息，请单击 Help 按钮。若要获取某项具体操作的更多信息，请单击 QuickStart Guide 中对应的条目。

## 在线文档

在线帮助系统是您的主要信息来源。您可以随时通过选择 File: Help  Help Contents  访问帮助系统的概览页面。在线帮助系统包含功能强大的全文搜索引擎。

在每个对话框中，都有一个专用的 Help 按钮，可直接打开相应的手册页面。此外，当某个特定模式处于活动状态时，F1 键可提供上下文相关帮助。例如，当选中一个块时按 F1 键，您将获得有关该块属性的信息。

如果没有可用的特定信息，按 F1 键将打开一个概览页面，您可以从该页面浏览帮助系统。

关于 CST Studio Suite 在线文档使用方法的更详细说明，请参阅《CST Studio Suite - Getting Started》手册。

## 教程和示例

组件库提供了教程和示例；当您尝试解决某个具体问题时，它们通常是首选的信息来源。另请参阅在线帮助系统起始页中 Tutorials and Examples 链接下给出的说明。我们建议您浏览所有可用教程和示例的列表，并选择最接近您应用场景的内容。

## 技术支持

在联系技术支持之前，您应先查阅在线帮助系统。如果这仍无法帮助您解决问题，可以在知识库中查找更多信息，并通过 3DS.com/Support 获取一般产品支持。

## 宏语言文档

有关特定模块内置宏语言的更多信息，可以在在线帮助系统的 VBA 书籍 Visual Basic (VBA) Language 中访问。宏语言文档由四部分组成：

 宏语言的概述和一般说明  
 所有特定宏语言扩展的说明。  
 与 Visual Basic for Applications (VBA) 兼容的宏语言语法参考。  
 一些带文档说明的宏示例

## 变更历史

可以通过帮助系统主页上的 What’s New in this Version 链接，或通过 File: Help 后台页面，查看软件最新版本中重要变更的概览。由于每个新版本都会包含许多新功能，即使您已经熟悉以前的某个版本，也应浏览这些列表。
